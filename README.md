## Compilation & Execution

To play the game
```
make re && ./fractol julia
```
or
```
make re && ./fractol mandelbrot
```

## Instructions

* ```WASD``` to move
* ```J/K``` Increase/Reduce iterations
* ```-/= or Mousewheel``` Zoom
* ```ESC``` to quit
